Carga masiva (una sola vez)

Pegá varias categorías y productos de una. Una línea que empieza con # es una categoría (el tipo de producto: Tinte, Oxidante, Decolorante, etc.); las líneas siguientes son productos de esa categoría, en formato nombre; marca; línea; tono; formato; proveedor; código (todo menos el nombre es opcional, se puede dejar vacío entre punto y coma). Ejemplo: 

# Tinte
Color Touch 5/0 Castaño Claro; Wella; Color Touch; 5/0; 60 ml
Illumina 9/60 Rubio Muy Claro Dorado; Wella; Illumina Color; 9/60; 60 ml

# Oxidante
Welloxon Perfect 20 vol; Wella; Welloxon Perfect;; 1000 ml
